# Deploy Alternativo - Vercel (GRÁTIS)

## Vercel é mais fácil que GitHub Pages!

### Passos:

1. **Acesse:** https://vercel.com
2. **Clique em "Sign Up"**
3. **"Continue with GitHub"**
4. **Autorize o GitHub**
5. **Importe seu repositório:** fluxo-caixa-profissional
6. **Clique em "Deploy"**
7. **Pronto! Site no ar em 1 minuto**

### URL será:
- https://fluxo-caixa-profissional-xxx.vercel.app
- (xxx são números aleatórios)

### Vantagens:
- ✅ Mais rápido que GitHub Pages
- ✅ Deploy automático
- ✅ GRÁTIS sem limites
- ✅ Suporte a backend

### Se preferir corrigir GitHub Pages:
1. Verifique se o index.html está na raiz
2. Verifique se a pasta js/ está completa
3. Aguarde mais 5 minutos
4. Acesse: https://salemlavarapido.github.io/fluxo-caixa-profissional/
